CoCreate-fullscreen

Toggle any html element to view in fullscreen mode. Easy configuration and targeting using the fullscreen and target attribute.

Install

 
npm i @cocreate/fullscreen

Or you can use cdn link:

<script>https://cdn.cocreate.app/fullscreen/latest/CoCreate-fullscreen.min.js</script>

Usage

This is fullscreen usage

                                
                                      <style type="text/css">
                                        #example2{
                                          background-color: white;
                                          }
                                      </style>
                                      <section>
                                        <h2>Only Target</h2>
                                        <button fullscreen target="#example2">With Attr Target</button>
                                        <button fullscreen >WithOut Target</button>
                                      </section>
                                      <section>
                                        <div id="example2" class="box ">
                                          <h2>#exemple2</h2>
                                            <p>
                                            Lorem ipsum dolor sit amet, consectetur adipiscing elit.
                                            </p>
                                        </div>
                                      </section>

This is fullscreen usage
